# Tags and the API

## Researching the Tags and API calls in Flickr

It appears that the API calls to flickr do not have a call to generate a single list of all the tags used on photos under a single user account.

### How to get the tags

Get each image and get the tags from each image.


### Examples

* flickr.commons.getInstitutions
